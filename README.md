## ğŸ“Œ Project Overview
Peak electricity demand leads to grid instability, high operational costs, and increased carbon emissions. Traditional power grids lack real-time consumer-side demand management.

This project proposes a Cloud-Based Industrial IoT (IIoT) Smart Grid Demand-Side Management Platform that dynamically adjusts non-essential loads based on time-of-use pricing.

The system simulates smart meter data, applies dynamic pricing signals, and automatically manages devices such as EV chargers and air conditioners during peak hours.
